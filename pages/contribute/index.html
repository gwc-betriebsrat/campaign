Contribute to the Website

The GWC website is maintained by the team — not just engineers. This page explains how to get access, where everything lives, and how to make changes, even if you've never touched code before.

What is GitHub?

GitHub is where all the website files are stored and edited. Think of it like a shared Google Drive — but every change is tracked, nothing is ever permanently lost, and updates go live automatically within about 60 seconds of saving.

You don't need to be a software engineer. If you can edit a Word document, you can make text changes to this website directly in your browser — no software to install.

Getting started

1

Create a GitHub account

Go to github.com and sign up for a free account. It takes two minutes.

2

Request access

Share your GitHub username with a team admin. They'll add you as a collaborator on the repository.

3

Read the guide

Open CONTRIBUTING.md in the repo — it walks you through everything step by step.
